# Прототип системы диагностики ЭКГ
Перед запуском убедитесь, что в папке ml извлечена папка data, она сжата т.к. в ней хранятся основные данные для обучения моделей.
В папке ml/models реализовано обучение ансамблевой модели классификаторов, состоящей из деревьев решений, snn, knn. 

После обучения состояние моделей сохраняется и для работы с пользователем используется предобученная модель, что позволяет ускорить процесс получения результата.

Рекомендованная конфигурация:

pandas>=1.3.0,

numpy>=1.21.0,

scikit-learn>=1.0.0,

matplotlib>=3.4.0,

joblib>=1.0.0,

openpyxl>=3.0.0
